# NeuroPrep Frontend

Modern SaaS-style React frontend for the AI Study Companion Platform.

## Tech
- React + Vite + TypeScript
- TailwindCSS
- Framer Motion
- Chart.js + react-chartjs-2
- React Router

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages
- Landing page
- Dashboard home
- AI Tutor
- Notes
- Planner
- Mock Tests
- Analytics
- Games
- Timer
- Settings
