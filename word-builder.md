# NeuroPrep Word Builder Specification

## Overview
Word Builder is a premium brain-training game where players unscramble glowing letter tiles to form words.

## Modes
- Classic: unlimited time
- Speed: 60-second timed run
- Survival: one wrong answer ends the run
- Battle: race against a bot score feed

## Categories
- Science
- Technology
- Programming
- English
- History
- Mathematics
- General Knowledge

## Sample Puzzles
- PTACMOEUR -> COMPUTER
- TNYHOP -> PYTHON
- VCIAJSRAPT -> JAVASCRIPT
- TAADBSAE -> DATABASE
- LMGHORAIT -> ALGORITHM
- NFUCTION -> FUNCTION
- RAAVY -> ARRAY
- JBTOCE -> OBJECT

## Progression
- 100 XP per correct answer
- 100 XP per level
- streak tracking
- combo multiplier
- accuracy percentage
- 10 coin reward per correct answer

## Mechanics
- drag-and-drop tile input with tap fallback
- shuffle letters
- hint costs 50 coins
- skip puzzle
- real-time timer updates
- session leaderboard
- level-up animation
- success particle burst
